# Wymagane wiadomości

Celem ćwiczenia jest omówienie zagadnień związanych z programowaniem gniazd TCP i UDP. Bazą dla ich przedstawienia będą programy napisane w językach Python i Java. Szczególny nacisk zostanie położony na przedstawienie problemów komunikacyjnych istniejących na styku różnych platform sprzętowych i programowych.

Wymagane wiadomości:
- protokoły TCP oraz UDP (sposób działania, cechy charakterystyczne, zastosowanie),
- asocjacje, gniazda, porty,
- reprezentacja danych różnych typów (kodowanie, liczba i kolejność bajtów).

Przed zajęciami należy zapoznać się także z podstawami programowania interfejsu gniazd w językach Python i Java:
- Java Sockets: https://docs.oracle.com/javase/tutorial/networking/sockets/index.html
- Python Sockets: https://docs.python.org/3/howto/sockets.html

Ćwiczenia można wykonywać na maszynach dostępnych w laboratorium albo na własnym komputerze (należy wcześniej zainstalować środowisko do programowania w językach Python i Java).
